# confshelf

confshelf is a just helper for managing dotfiles.

## Requirements

- Nim (Tested with version 1.4.4)

## Installation

Install Nim if you haven't installed yet. ([Nim Install Instruction](https://nim-lang.org/install.html))

```sh
mkdir dotfiles
cd dotfiles
wget -O setup.nims https://raw.githubusercontent.com/kuro46/confshelf/main/setup.nims
# In curl:
#curl -Lo setup.nims https://raw.githubusercontent.com/kuro46/confshelf/main/setup.nims
chmod +x ./setup.nims
./setup.nims --create-config
```

Then follow instructions in links.conf (Generated by `--create-config`)
