# confshelf

confshelf is a helper for managing dotfiles.

## Requirements

- Nim (Tested with version 1.4.4)

## Installation

Install Nim if you haven't installed yet. ([Nim Install Instruction](https://nim-lang.org/install.html))

```sh
mkdir dotfiles
cd dotfiles
wget -O confshelf.nims https://raw.githubusercontent.com/kuro46/confshelf/main/confshelf.nims
# In curl:
#curl -Lo confshelf.nims https://raw.githubusercontent.com/kuro46/confshelf/main/confshelf.nims
chmod +x ./confshelf.nims
./confshelf.nims --create-config
```

Then follow instructions in links.conf (Generated by `--create-config`)
